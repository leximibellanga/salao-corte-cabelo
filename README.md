# salao-corte-cabelo
Este projeto é um sistema de agendamentos e gestão para salões e barbearias, permitindo que os clientes marquem serviços online e os administradores controlem agendamentos, serviços, clientes e ganhos. O objetivo é oferecer uma plataforma prática, moderna e responsiva, que facilite o atendimento e a organização do salão.
Link do site:  https://leximibellanga.github.io/salao-corte-cabelo/
